# Wallet Service

Простой сервис для управления кошельками с операциями пополнения и снятия средств.

## Описание

Проект реализует REST API для управления кошельками:
- Получение баланса кошелька
- Пополнение баланса
- Снятие средств с кошелька

## Технологии

- Java 17
- Spring Boot 3
- PostgreSQL
- Maven
